🔧 MODIFICATIONS DANS PACKAGE.JSON
═══════════════════════════════════════════════════════════════

Ajouts :

1. Métadonnées Electron
   ─────────────────────
   • name: "geset-pro"
   • productName: "GESET Pro"
   • description: "Système de Gestion Scolaire"
   • author: "GESET Pro"
   • main: "electron/main.js"
   • version: "1.0.0"

2. Scripts Electron
   ─────────────────────
   • electron:dev          → Développement Electron
   • electron:build        → Build toutes plateformes
   • electron:build:win    → Build Windows uniquement
   • electron:build:mac    → Build macOS uniquement
   • electron:build:linux  → Build Linux uniquement

3. Configuration electron-builder
   ────────────────────────────────
   • appId: "com.gesetpro.app"
   • Dossier de sortie: release/
   • Configuration Windows (NSIS)
   • Configuration macOS (DMG)
   • Configuration Linux (AppImage, DEB)

4. Dépendances ajoutées
   ─────────────────────
   • electron
   • electron-builder
   • concurrently
   • wait-on
   • cross-env

🔧 MODIFICATIONS DANS VITE.CONFIG.TS
═══════════════════════════════════════════════════════════════

Ajouts :

• base: './'              → Chemins relatifs pour Electron
• outDir: 'dist'          → Dossier de sortie explicite
• emptyOutDir: true       → Nettoyage avant build
• server.port: 5173       → Port fixe pour dev
• server.strictPort: true → Pas de port alternatif

📊 RÉSULTAT FINAL
═══════════════════════════════════════════════════════════════

Après build, vous obtiendrez dans release/ :

Windows :
  • GESET-Pro-Setup-1.0.0.exe    (installateur NSIS)
  • win-unpacked/                 (version portable)

macOS :
  • GESET-Pro-1.0.0.dmg          (image disque)

Linux :
  • GESET-Pro-1.0.0.AppImage     (exécutable universel)
  • geset-pro_1.0.0_amd64.deb    (paquet Debian/Ubuntu)


═══════════════════════════════════════════════════════════════
🎯 UTILISATION
═══════════════════════════════════════════════════════════════

Pour créer l'exécutable Windows :
  1. Ouvrir un terminal
  2. Taper : npm install
  3. Taper : npm run electron:build:win
  4. Récupérer dans release/

OU

  Double-clic sur : build-windows.bat
