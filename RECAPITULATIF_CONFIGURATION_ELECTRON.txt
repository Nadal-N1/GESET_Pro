🚀 COMMANDES PRINCIPALES
═══════════════════════════════════════════════════════════════

Développement Web :
  npm run dev              → Lance le serveur web

Développement Electron :
  npm run electron:dev     → Lance l'appli desktop en dev

Créer Installateur Windows :
  npm run electron:build:win

Créer Installateur macOS :
  npm run electron:build:mac

Créer Installateur Linux :
  npm run electron:build:linux

📈 PROCHAINES ÉTAPES SUGGÉRÉES
═══════════════════════════════════════════════════════════════

1. Créer le premier exécutable :
   npm run electron:build:win

2. Tester l'installateur sur une machine propre

3. Préparer la distribution :
   - Créer un site de téléchargement
   - Préparer un guide utilisateur
   - Créer un changelog

4. Optionnel : Signer le code
   - Obtenir un certificat de signature
   - Configurer dans package.json
   - Éviter les avertissements Windows

5. Optionnel : Auto-update
   - Configurer GitHub releases
   - Activer les mises à jour automatiques


═══════════════════════════════════════════════════════════════
💡 CONSEILS IMPORTANTS
═══════════════════════════════════════════════════════════════

• Le premier build est long (~10-15 min), c'est normal
• Toujours tester le build web avant de créer l'exécutable
• L'exécutable inclut Chromium (~70 MB), c'est normal
• Testez sur différentes machines Windows avant distribution
• Changez la version dans package.json pour chaque release
• Gardez un backup de chaque version créée
• Documentez les changements dans un changelog
