  GESET Pro - Créer l'Exécutable Windows
  Guide Ultra-Rapide
═══════════════════════════════════════════════════════════════

🎯 EN 3 ÉTAPES SIMPLES
═══════════════════════════════════════════════════════════════

1️⃣  INSTALLER LES DÉPENDANCES
   ───────────────────────────
   Ouvrez un terminal et tapez :

   npm install


2️⃣  CRÉER L'INSTALLATEUR
   ───────────────────────────
   Sur Windows, double-cliquez sur :

   build-windows.bat

   OU tapez dans le terminal :

   npm run electron:build:win


3️⃣  RÉCUPÉRER LE FICHIER
   ───────────────────────────
   L'installateur est créé dans le dossier :

   release/GESET-Pro-Setup-1.0.0.exe

   C'est ce fichier que vous distribuez !


═══════════════════════════════════════════════════════════════
📋 PRÉREQUIS
═══════════════════════════════════════════════════════════════

Vous devez avoir installé :
  ✓ Node.js 18 ou plus récent (https://nodejs.org)
  ✓ Windows 10 ou 11 (pour build Windows)


═══════════════════════════════════════════════════════════════
⏱️  DURÉE
═══════════════════════════════════════════════════════════════

• Installation des dépendances : 2-5 minutes
• Création de l'installateur  : 5-10 minutes
• TOTAL                        : ~10-15 minutes

📞 COMMANDES UTILES
═══════════════════════════════════════════════════════════════

Développement :
  npm run dev                  # Lancer en dev (web)
  npm run electron:dev         # Lancer en dev (Electron)

Production :
  npm run build                # Compiler l'app web
  npm run electron:build:win   # Créer installateur Windows
  npm run electron:build:mac   # Créer installateur macOS
  npm run electron:build:linux # Créer installateur Linux

Nettoyage :
  rm -rf node_modules dist release
  npm install                  # Réinstaller tout
